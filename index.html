Simon Game

Click the 'Start' button to begin a new game. The computer will display a flashing color which you must then match by clicking one of the color buttons. This will continue - each turn increasing in length of the color sequence - until you incorrectly match the computer's sequence. At any point you can choose to start over with a new game by pressing the 'New Game' button.
